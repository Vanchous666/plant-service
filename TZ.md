# Техническое задание

## Название проекта
Сервис подборки комнатных растений для городских жителей с учётом условий жилья

## Цель
Создание системы подбора комнатных растений на основе анализа условий проживания и возможностей ухода пользователя.

## Функциональные требования
1. Регистрация и авторизация пользователей
2. Ввод параметров комнаты (освещение, температура, влажность)
3. Каталог растений с фильтрацией
4. Алгоритм подбора растений по совместимости
5. Учет растений пользователя
6. Система напоминаний по уходу
7. Аналитика и статистика

## Модели данных
1. **Plant** - информация о растении
2. **RoomCondition** - условия комнаты пользователя
3. **UserPlant** - растения пользователя
4. **CareReminder** - напоминания по уходу
5. **PlantRecommendation** - рекомендации растений

## Стек технологий
- Django 6.0.1
- Bootstrap
- SQLite/PostgreSQL
- Pandas/Matplotlib для аналитики
